# ⚡ EXÉCUTER database.sql - 2 Minutes

## Les 7 Étapes (Aucune explication, juste faire):

1. **Dans Replit**
   - Ouvrir `database.sql`
   - `Ctrl+A` → `Ctrl+C`

2. **Aller sur Supabase**
   - Dashboard → Votre projet
   - Menu gauche → **SQL Editor**

3. **Cliquer "New Query"**

4. **Dans l'éditeur**
   - `Ctrl+V` (coller)

5. **Cliquer le bouton bleu "Run"**

6. **Attendre le message ✓**
   ```
   Success! Ran X commands.
   ```

7. **Vérifier les tables**
   - Table Editor → Vous devez voir 10 tables
